Introduction

The plant identification project aims to provide users with a convenient way to identify different plant species using Plant Identfication API.

Arduino Uno

The Arduino Uno is a microcontroller board based on the ATmega328P. It is used in this project for programming the ESP32-CAM module because it does not have usb interface.

Features:

  • Microcontroller: ATmega328P
  • Digital I/O Pins: 14 (of which 6 provide PWM output)
  • Analog Input Pins: 6
  • Flash Memory: 32 KB (0.5 KB used by bootloader)
  • SRAM: 2 KB
  • EEPROM: 1 KB

ESP32 Cam Module

The ESP32 Cam module is a tiny ESP32-based chip that comes with an OV2640 camera and microSD card slot. It is used in this project for capturing plant images.

Features:

  • Microcontroller: ESP32
  • Camera: OV2640
  • Integrated Wi-Fi and Bluetooth
  • Supports microSD card
  • Compact size

Breadboard

A breadboard is a reusable solderless device used to build and test electronic circuits. It provides a platform for prototyping and experimenting with different components in the project.
